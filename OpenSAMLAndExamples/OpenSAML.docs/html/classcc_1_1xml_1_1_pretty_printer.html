Detailed Description

This class "pretty prints" an XML stream to something more human-readable. It duplicates the character content with some modifications to whitespace, restoring line breaks and a simple pattern of indenting child elements.

This version of the class acts as a SAX 2.0 DefaultHandler, so to provide the unformatted XML just pass a new instance to a SAX parser. Its output is via the toString method.

One major limitation: we gather character data for elements in a single buffer, so mixed-content documents will lose a lot of data! This works best with data-centric documents where elements either have single values or child elements, but not both.
